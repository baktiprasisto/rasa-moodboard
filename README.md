# 🌙 Rasa — Moodboard Harian

Rasa is a **minimalist daily mood tracker web app** that helps you record your emotions, energy level, and daily experiences in a clean and aesthetic interface.

Designed with a **calm dark UI**, this project focuses on simplicity, emotional awareness, and beautiful interaction.

---

## 🚀 Live Demo

👉 https://rasa-moodboard.netlify.app/

---

## ✨ Features

* 🎭 Select your daily mood (Bahagia, Tenang, Galau, etc.)
* 📝 Write short notes about your day
* 🏷️ Add tags (work, family, health, etc.)
* ⚡ Track your energy level with slider
* 📅 Weekly mood visualization (mood map)
* 🕘 Automatically saves last 5 entries
* 💾 Data stored in browser using **localStorage**

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Custom UI + Responsive Design)**
* **Vanilla JavaScript**
* **LocalStorage API**

---

## 🧠 How It Works

* User selects a mood and optional tags
* Adds a short note and energy level
* Clicks **Save**
* Data is stored in `localStorage`
* The app automatically:

  * Updates weekly mood map
  * Displays latest entries
  * Formats date dynamically
